# GraphQL Upload
Drupal module to upload files with the GraphQL module.

## Compatibility
Tested on [GraphQL 8.x-3.0-beta1](https://github.com/drupal-graphql/graphql/tree/8.x-3.0-beta1).

### [!] WARNING
This is a sandbox module, so use it on your own risk.
